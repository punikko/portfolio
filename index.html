Web Coder

プログラマーからWebコーダーへ転身。

プログラマーとしての開発経験を活かし、徹底した品質チェックや表示テストを行うことで、手戻りの少ない高品質な実装を提供します。
また、「後から他の人が見ても読みやすいコード」や「保守性の高いCSS設計（BEMなど）」を意識し、デザインの意図を正確に再現したレスポンシブWebサイトを制作いたします。

Skills

HTML5 / CSS3

BEMなどのCSS設計手法を用いた保守性の高いマークアップ、Flexbox/CSS Gridを活用したレスポンシブWebデザインの構築が可能です。

JavaScript

品質管理 / ツール

WordPress
